# To-Do List

## Technology Stack / 技術スタック

+ Next.js 13 (App Router)
+ TypeScript
+ Material UI
+ MongoDB

## Quick Start(Dev)

```sh
# Set local env variable
cp .env.example .env.local

# Run MongoDB, MongoDB's admin board http://127.0.0.1:8081/ user/password(demo/demo)
docker-compose up -d

# Run Serve
npm run dev
```
